# OnlyFans Chat - Project Brief

## Обзор проекта
OnlyFans Chat - это веб-приложение, разработанное для имитации чата OnlyFans с поддержкой отправки сообщений и медиафайлов. Приложение построено на Next.js с использованием React и TypeScript.

## Ключевые цели
- Создать интуитивно понятный и отзывчивый интерфейс чата
- Обеспечить поддержку отправки текстовых сообщений и медиафайлов
- Реализовать все основные функции чата OnlyFans
- Создать приложение, которое работает быстро и надежно

## Основные требования
1. Авторизация пользователей
2. Отображение списка чатов
3. Выбор и переключение между чатами
4. Отправка и получение текстовых сообщений
5. Загрузка и отправка медиафайлов
6. Возможность установки цены на медиафайлы
7. Поддержка прокрутки истории сообщений
8. Отображение статусов сообщений (отправлено, доставлено, прочитано)

## Технический стек
- Frontend: Next.js, React, TypeScript
- Styling: Tailwind CSS
- UI компоненты: сочетание собственных компонентов и библиотек Radix UI
- Иконки: Heroicons, Lucide React

## Текущий статус
Базовый прототип приложения разработан и включает основные функции чата. Требуется дальнейшая доработка и улучшение пользовательского интерфейса.
